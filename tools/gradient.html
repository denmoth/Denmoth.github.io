How it works

This tool interpolates colors between the Start and End points for each character in your text.

Supported by Spigot/Paper servers and many mods using the &#rrggbb format.

Как это работает

Инструмент рассчитывает промежуточные цвета для каждой буквы текста.

Формат &#rrggbb поддерживается на серверах Paper/Spigot и в современных модах.
