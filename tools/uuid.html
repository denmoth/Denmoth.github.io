Generate offline (random) UUIDs for players or entities.

Генерация оффлайн (случайных) UUID для игроков или сущностей.

Usage Guide

This tool generates a Version 4 (Random) UUID.

  • Entities: Assign specific UUIDs to mobs in summon commands to target them later.
  • Attribute Modifiers: Items with custom attributes require unique UUIDs for each modifier.
  • Map Making: Use fixed UUIDs for boss bars or persistent entities.

Гайд

Этот инструмент создает UUID Версии 4 (Случайный).

  • Сущности: Используйте для призыва мобов с фиксированным ID, чтобы потом находить их командами.
  • Атрибуты: Для предметов с кастомными атрибутами нужны уникальные UUID для каждого модификатора.
  • Картостроение: Фиксированные UUID нужны для боссбаров и постоянных NPC.
