What is a Minecraft UUID?

A UUID (Universally Unique Identifier) is a 128-bit label used to uniquely identify objects. In Minecraft, they are used for:

  • Entities: Persistent mobs that map makers can target with commands (e.g., boss bars).
  • Attributes: Items with custom modifiers (damage, speed) need unique UUIDs to stack correctly.
  • Player Data: While online players use Mojang UUIDs, offline players or NPCs use random ones generated here.
{% endlang %} {% lang ru %}

Что такое UUID в Minecraft?

UUID (Универсальный Уникальный Идентификатор) — это 128-битная метка. В Майнкрафте они используются для:

  • Сущностей: Постоянные мобы, которых картоделы могут находить командами (например, для боссбаров).
  • Атрибутов: Предметы с кастомными модификаторами (урон, скорость) требуют уникальные UUID, чтобы корректно стакаться.
  • Игроков: Оффлайн игроки или NPC используют случайные UUID, которые генерирует этот инструмент.
