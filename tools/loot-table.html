Basic Block Loot Table Generator.

Базовый генератор лута для блоков.

Guide

Generates a standard loot table for a block.

  • Silk Touch: If checked, item only drops with Silk Touch.
  • Fortune: Applies standard fortune multiplier formula.

Гайд

Генерирует стандартную таблицу лута для блока.

  • Silk Touch: Если выбрано, предмет выпадает только с Шелковым касанием.
  • Fortune: Применяет стандартную формулу удачи.
