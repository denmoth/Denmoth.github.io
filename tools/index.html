About These Tools

This collection of developer tools is designed to speed up Minecraft modding workflow. All generators run locally in your browser, ensuring privacy and speed.

Popular Tools:

  • UUID Generator: Creates Version 4 UUIDs for entities, attribute modifiers, and map making.
  • JSON Text: Visual builder for /tellraw and /title commands with click/hover events.
  • Pack.mcmeta: Updates pack formats for 1.21, 1.20.4, and older versions instantly.
{% endlang %} {% lang ru %}

Об инструментах

Эта коллекция инструментов создана для ускорения разработки модов и карт. Все генераторы работают локально в вашем браузере, обеспечивая приватность и скорость.

Популярное:

  • UUID Generator: Создание UUID Версии 4 для сущностей, атрибутов и картостроения.
  • JSON Text: Визуальный конструктор команд /tellraw и /title с поддержкой кликов.
  • Pack.mcmeta: Быстрое обновление форматов паков для 1.21, 1.20.4 и старых версий.
