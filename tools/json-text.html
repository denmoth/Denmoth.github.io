How to use

Basic Formatting

  • Select a color using the picker.
  • Use buttons B, I, U, S to toggle styles.

Events

  • Click Event: Triggers when a player clicks the text in chat. run_command executes immediately.
  • Hover Event: Shows a tooltip when hovering.

Как использовать

Форматирование

  • Выберите цвет через палитру.
  • Используйте кнопки B, I, U, S для стилей.

События

  • Click Event: Срабатывает при клике. suggest_command подставляет команду в чат.
  • Hover Event: Показывает подсказку при наведении.
