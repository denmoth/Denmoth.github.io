Theming

Темизация

{% endlang %} {% lang en %}

CubeUI uses ThemeManager to handle rendering styles. The default theme is Vanilla-like dark.

{% endlang %} {% lang ru %}

CubeUI использует ThemeManager для стилизации. Стандартная тема — темная ванильная.
