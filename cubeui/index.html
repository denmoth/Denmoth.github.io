Темизация

{% endlang %} {% lang en %}

CubeUI uses ThemeManager to handle rendering styles. The default theme is Perfectly Vanilla.

{% endlang %} {% lang ru %}

CubeUI использует ThemeManager для стилизации. Стандартная тема — Идеально Ванильная.

CuiButton

{% lang en %}

Resizable button with 3-slice rendering (Left, Center, Right). Supports hover and disabled states.

{% endlang %} {% lang ru %}

Растягиваемая кнопка с 3-slice рендером. Поддерживает наведение и выключенное состояние.

{% endlang %}
Java 
this.addRenderableWidget(new CuiButton(x, y, 100, 20, Component.literal("Save"), btn -> {
    System.out.println("Clicked");
}));

CuiTextField

{% lang en %}

Replacement for vanilla EditBox. Supports custom background from Atlas and placeholder text.

{% endlang %} {% lang ru %}

Замена ванильному EditBox. Поддерживает фон из Атласа и плейсхолдер.

{% endlang %}
Java 
CuiTextField input = new CuiTextField(font, x, y, 150, 20, Component.empty());
input.setPlaceholder(Component.literal("Search..."));
this.addRenderableWidget(input);

CuiPanel

{% lang en %}

9-slice background panel for GUIs.

{% endlang %} {% lang ru %}

9-slice панель для фона интерфейсов.

{% endlang %}
Java 
this.addRenderableWidget(new CuiPanel(x, y, 200, 100));

Container Screen

{% lang en %}

Extends AbstractContainerScreen. Auto-renders slots and highlights using CuiRenderHelper.

{% endlang %} {% lang ru %}

Наследует AbstractContainerScreen. Автоматически рендерит слоты и подсветку через CuiRenderHelper.

{% endlang %}
Java 
public class MyGui extends CuiContainerScreen {
    public MyGui(MyMenu menu, Inventory inv, Component title) {
        super(menu, inv, title);
    }
}
