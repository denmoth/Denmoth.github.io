Library

CubeUI

Lightweight GUI library for Forge/NeoForge. Version 0.1.0.

Installation

Add to build.gradle:

implementation fg.deobf("com.denmoth:cubeui:0.1.0-beta")

Widgets

CuiButton

3-slice resizable button.

new CuiButton(x, y, 100, 20, Component.literal("Click"), btn -> {});

CuiTextField

Input field with placeholder support.

CuiTextField input = new CuiTextField(font, x, y, 100, 20, Component.empty());
input.setPlaceholder(Component.literal("Search..."));

CuiSlot

Visual item slot.

Note: Currently decorative only. Interaction updates coming in 0.2.0.
CuiSlot slot = new CuiSlot(x, y);
slot.setItem(new ItemStack(Items.DIAMOND));
