CuiButton

A resizable button with 3-slice rendering (Left Cap, Center Tile, Right Cap).

Java 
this.addRenderableWidget(new CuiButton(x, y, 100, 20, Component.literal("Save"), btn -> {
    // On Press Logic
}));

CuiTextField

Replacement for EditBox. Supports custom backgrounds via Atlas and placeholders.

Java 
CuiTextField input = new CuiTextField(font, x, y, 150, 20, Component.empty());
input.setPlaceholder(Component.literal("Search..."));
this.addRenderableWidget(input);

CuiSlot

Decorative item slot. Note: For interactive inventory, use CuiContainerScreen.

Java 
CuiSlot slot = new CuiSlot(x, y);
slot.setItem(new ItemStack(Items.DIAMOND));
this.addRenderableWidget(slot);

Resource Pack Guide

CubeUI uses a single atlas texture located at assets/cubeui/textures/gui/gui_components.png.

  • Panel (9-slice): 0,0 to 10,10 (Corner size: 4px)
  • Button (Normal): 22,0 (Left), 25,0 (Center), 42,0 (Right)
  • TextField: Uses specific UVs defined in VanillaTheme.java.
