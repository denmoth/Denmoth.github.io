# CubeUI

**Version:** 0.1.0-beta
**License:** MIT
**Source:** [GitHub Repository](https://github.com/Denmoth/CubeUI)

CubeUI provides a set of tools for developers to create responsive and scalable user interfaces in Minecraft without the boilerplate of vanilla rendering methods.

## Key Features

* **Rendering:** 9-slice scaling for windows and panels.
* **Widgets:** Custom buttons, text fields with placeholders, slots.
* **Integration:** Seamless linking with `AbstractContainerMenu`.
* **Theming:** Texture atlas support via `VanillaTheme`.
